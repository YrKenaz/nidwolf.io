## 파일별 역할 요약

### 사이트 파일

| 파일 | 크기 | 설명 |
|------|------|------|
| `design-toolkit.html` | ~246KB | 메인 도구 앱. 사이드바 + 콘텐츠 영역. 전체 기능 인라인 |
| `ladder.html` | ~40KB | 사다리타기 추첨. Canvas 기반, 스킨 시스템 분리 |
| `articles/style.css` | - | 아티클 전용 CSS. key-points, inline-related 포함 |
| `sitemap.xml` | - | 44개 URL. 우선순위: 메인 1.0 → 아티클 0.8 |
| `robots.txt` | - | `Allow: /` + Sitemap 경로 |

### 개발 파일 (`_dev/`)

| 파일 | 설명 |
|------|------|
| `build.py` | 아티클 + index + about + privacy 생성. `python build.py` 실행 |
| `tips*.py` | 디자인 팁 데이터 (slug, title, desc, body, cat, icon) |
| `hist.py` | 디자인 역사 10편 데이터 |
| `*_en.py` | 영문 번역본 (title_en, desc_en, body_en) |
| `extras*.py` | 각 아티클 추가 본문 (1500자 미만 경고 방지용) |
| `static_pages.py` | about / privacy 페이지 생성 함수 |

## 경로 규칙

- 아티클 내부 `css_prefix = "../"` (articles/ 하위이므로 한 단계 상위)
- `build.py`의 `ROOT = normpath(join(__file__, '../..'))` 로 프로젝트 루트 참조
- `BASE_URL = "https://nidwolf.com"` 상수로 canonical / OG URL 생성
