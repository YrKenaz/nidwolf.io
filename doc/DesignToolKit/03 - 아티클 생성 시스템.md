## 아티클 루프 패턴

f-string 내 dict 리터럴 충돌을 막기 위해 **모든 변수를 f-string 밖에서 사전 계산**.

```python
for i, a in enumerate(items):
    en = EN.get(a['slug'], {})
    # ... 기타 변수 ...

    # ✅ f-string 밖에서 먼저 계산
    rel0_slug = rel3[0]["slug"]  if rel3 else ""
    rel0_ko   = rel3[0]["title"] if rel3 else ""
    rel0_en   = EN.get(rel0_slug, {}).get("title_en", rel0_ko) if rel3 else ""
    kp_box    = key_points_box(body_ko, body_en_text)
    jld       = jsonld_article(a['title'], a['desc'], a['slug'])
    art_url   = f"{BASE_URL}/articles/{a['slug']}.html"

    # ✅ f-string 내 {{{}}} 없이 깔끔하게 사용
    html = head(..., jsonld=jld, page_url=art_url) + header("../") + f"""
    ...{kp_box}...
    """
```

> ⚠️ f-string 안에서 `EN.get(x, {{}})` 같은 dict 리터럴 사용 시 `TypeError: unhashable type: 'dict'` 발생. 반드시 밖에서 계산할 것.

## 아티클 품질 기준

- **본문 최소 1,500자** (plain text 기준). 미달 시 실행 후 경고 출력.
- `extras_tips.py`, `extras2.py`로 보충 본문 추가 가능.
- key-points 박스: `<h2>` 태그 개수 = 목차 항목 수.
