## 신청 전 체크리스트

### 🔴 필수 (신청 전 반드시 완료)

- [ ] **사이트 배포**: `nidwolf.io` 실제 운영 (HTTPS 포함)
- [ ] **Google Search Console 등록**: 사이트 소유권 인증 + sitemap 제출
- [ ] **쿠키 동의 배너**: 하단 또는 팝업 형태로 쿠키 사용 안내 + 동의 버튼
- [ ] **트래픽 축적**: 최소 1~2개월 검색 유입 후 신청 (일 50~100 UV 이상 권장)

### 🟡 권장 (심사 통과율 향상)

- [ ] **아티클 대표 이미지 추가**: 각 아티클에 SVG 또는 PNG 삽입 (og:image용)
- [ ] **404 오류 페이지**: `nidwolf.io/404.html` 또는 서버 설정
- [ ] **Google Analytics 연동**: 트래픽 데이터 확보 (심사관 참고용)
- [ ] **SNS 채널 연결**: Twitter/Instagram 등 공식 채널 링크
- [ ] **콘텐츠 추가**: 아티클 40편 → 60편 이상으로 확대

### 🟢 이미 완료

- [x] 소개·연락처 페이지

## 단계별 신청 로드맵

```
현재
  │
  ▼
[1단계] 배포 (즉시)
  → nidwolf.io HTTPS 운영 시작
  → Google Search Console 등록 + sitemap 제출
  → 쿠키 동의 배너 추가
  │
  ▼ (1개월 후)
[2단계] 트래픽 확인
  → Search Console 에서 주간 클릭 50+ 확인
  → Google Analytics 설치 및 데이터 축적
  → 404 페이지 추가
  │
  ▼ (2~3개월 후)
[3단계] AdSense 신청 ← 권장 시점
  → 신청 → 심사 2~4주 → 승인 후 광고 코드 삽입
  → design-toolkit.html / articles/ 광고 위치 결정
```
