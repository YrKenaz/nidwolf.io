## 새 이슈 등록 방법

1. `_TEMPLATE.md`를 복사해 `BUG-XXX.md` 또는 `FEAT-XXX.md`로 저장
2. frontmatter(`id`, `type`, `status`, `severity`, `created`, `file`) 채우기
3. 본문(요약·증상·원인·수정) 작성
4. 아래 테이블에 한 줄 추가

## 이슈 ID 채번 규칙

| 접두어 | 용도 | 다음 번호 |
|--------|------|-----------|
| `BUG-` | 버그 / 예상치 못한 동작 | **BUG-005** |
| `FEAT-` | 기능 요청 / 개선 제안 | **FEAT-001** |
| `TD-` | 기술 부채 | **TD-001** |

---

## 심각도 기준

| 레벨 | 기준 |
|------|------|
| **Critical** | 게임/사이트 전체 불능, 데이터 손실 |
| **High** | 핵심 기능 동작 불가, 사용자에게 명확히 보이는 버그 |
| **Medium** | 기능 일부 불량, 회피책 있음 |
| **Low** | 경미한 시각 오류, 개발 편의성 문제 |
